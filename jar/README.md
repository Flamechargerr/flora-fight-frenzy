## How to Run

### Prerequisites
- Java 11 or higher
- JavaFX SDK 11 or higher

### Steps

1. Download JavaFX SDK from [Gluon](https://gluonhq.com/products/javafx/)
2. Extract the JavaFX SDK
3. Run the following command:

```bash
java --module-path /path/to/javafx-sdk/lib --add-modules javafx.controls,javafx.fxml,javafx.graphics,javafx.media -jar PlantVsZombies.jar
```

### Windows Example
```powershell
java --module-path "C:\javafx-sdk-17.0.2\lib" --add-modules javafx.controls,javafx.fxml,javafx.graphics,javafx.media -jar PlantVsZombies.jar
```

### Linux/Mac Example
```bash
java --module-path ~/javafx-sdk-17.0.2/lib --add-modules javafx.controls,javafx.fxml,javafx.graphics,javafx.media -jar PlantVsZombies.jar
```

## Game Features
- 5 Levels with increasing difficulty
- Day and Night modes
- 6 Plants: Sunflower, Pea Shooter, Cherry Bomb, Walnut, Repeater, Jalapeno
- 3 Zombie types: Normal, Conehead, Buckethead
- Save/Load game functionality
- Shovel tool to remove plants
- Sound effects
