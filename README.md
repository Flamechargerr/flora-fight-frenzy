## 🧠 About the Project

**Flora Fight Frenzy** is a fun and educational 2D tower defense game inspired by the iconic *Plants vs Zombies*.  
Built using [Lovable.io](https://lovable.io), it brings together visual game-building and classic arcade logic.  
The objective is to defend your garden by strategically placing plants that repel incoming threats like invasive species and pollution monsters.

---

## 🎮 Gameplay Overview

- Place plants on the field to attack or block enemies
- Earn points for stopping invaders
- Survive increasingly difficult waves
- Strategic + fun gameplay for all ages

---

## ✨ Features

- 🖱️ Drag-and-drop interface using Lovable.io’s visual builder  
- 🧠 Combines real-world ecological awareness with simple mechanics  
- 🕹️ Game logic built with HTML/CSS/JS customization  
- 🌱 Enemies modeled as real environmental threats  
- 🔄 Replayable levels with escalating difficulty

## 🛠️ How to Run Locally

```bash
git clone https://github.com/Flamechargerr/flora-fight-frenzy.git
cd flora-fight-frenzy
```

- Now, just double-click `index.html` or open it in your preferred browser.  
✅ No installations, frameworks, or complex setup needed.
